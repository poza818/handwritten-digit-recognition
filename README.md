## 🖥️ What is handwritten-digit-recognition?

This application lets you recognize handwritten digits using a trained AI. You can either upload an image of a digit or draw one on the screen. The app then predicts what number you wrote. It shows you the prediction and how confident it is.

The app uses convolutional neural networks (CNN). This is a type of AI trained on thousands of digit images (MNIST dataset). It works quickly and gives results right away.

You do not need any programming knowledge. The application has a simple graphical interface to make it easy to use.

---

## 🎯 Main Features

- Upload a photo of a handwritten digit and get an instant prediction.
- Draw digits using a simple on-screen drawing tool.
- Displays the prediction along with a confidence score.
- Built with Python using TensorFlow for AI, OpenCV for image processing, and Tkinter for the interface.
- Runs on Windows without the need to install complex software.
- Supports common image formats (.jpg, .png, .bmp).
